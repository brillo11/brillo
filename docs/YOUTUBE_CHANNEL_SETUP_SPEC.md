# 유튜브 채널 시작 도우미 페이지 기능명세서

## 개요

유튜브 채널 개설을 위한 이름, 설명, 키워드, 프로필 이미지를 도와주는 조력자 페이지. 5단계의 워크플로우를 통해 채널명, 핸들명, 채널 설명, 채널 키워드, 프로필 이미지를 생성합니다.

## 브랜드 컬러

- **Primary**: Red-600 (`#DC2626`) / Orange (`#EA580C`)
- **Gradient**: `from-red-600 to-orange-600` (주요 버튼 및 CTA 요소)
- **Success**: Green (`#10B981`)
- **Warning**: Amber (`#F59E0B`)

## 전체 구조

- 상단에 5단계 진행 상황 표시 (Progress Bar)
- 단계: Channel Info → Channel Name → Handle → Description → Keywords → Profile Image

---

## Step 1: 채널 정보 입력 (Channel Info)

### 기능

- 채널 주제/장르 입력
- 하고 싶은 콘텐츠 설명
- 대상 시청자 정보 (선택 사항)
- 제출 시 Step 2로 이동

### 데이터

- 채널 주제
- 콘텐츠 설명
- 대상 시청자 (국가/언어)

---

## Step 2: 채널명 추천 (Channel Name)

### 기능

- 입력된 정보를 바탕으로 채널명 추천
- 채널명 추천 원칙:
  - 널리 전파될 가능성 고려
  - 기억하기 쉽고, 발음하기 쉽고, 검색하기 쉬운 이름
  - 독특하며 유명 브랜드명과 겹치지 않음
  - 검색엔진 오타 수정 가능 형태 피함
  - 최소 2자에서 최대 7자 미만 (띄어쓰기 없음)
  - 대상 언어에 맞춘 이름 (한국어/영어 등)
  - 한국어와 영어 혼합 비권장
- 여러 개의 채널명 후보 제시
- 사용자가 마음에 들지 않을 경우 재추천 가능
- 선택 후 검증 단계 안내

### 검증 단계

- 유튜브 검색 링크 제공 (https://www.youtube.com/)
- 정보성/사업 목적 채널의 경우 상표권 검색 안내 (키프리스: https://www.kipris.or.kr/khome/main.do)
- 검색 결과 확인 후 사용 가능 여부 판단
- 최종 채널명 선택

### 데이터

- 추천된 채널명 목록
- 선택된 채널명
- 검증 결과

---

## Step 3: 핸들명 추천 (Handle)

### 기능

- 채널명을 기반으로 핸들명 추천
- 핸들명 추천 원칙:
  - 채널명이 한글일 경우 핸들명도 한글로 유도
  - 채널 이름과 최대한 비슷하게 유지
  - 기억하기 쉬운 숫자(12시, 22 등)나 허용된 특수문자(\_ . - ·) 결합
  - 채널 이름 자체는 변형하지 않고 유지
- 5개의 후보 제시
- 유튜브 채널 생성 단계에서 실제 사용 가능 여부 검토 안내

### 데이터

- 추천된 핸들명 목록 (5개)
- 선택된 핸들명

---

## Step 4: 채널 설명 추천 (Description)

### 기능

- 초기 입력한 주제와 내용을 바탕으로 채널 설명 작성
- 채널 설명 추천 원칙:
  - 한 줄로 간략하고 직관적
  - 검색에 유용한 키워드 포함
  - 불필요하게 길거나 모호한 문장 피함
  - 특정 계층 직접 지칭 피함 (시니어, 노년층 등)
  - 보편적인 표현 사용
  - 해외 대상자일 경우 해당 국가 언어로 작성
- 5개의 후보 제시

### 데이터

- 추천된 채널 설명 목록 (5개)
- 선택된 채널 설명

---

## Step 5: 채널 키워드 추천 (Keywords)

### 기능

- 채널 주제와 콘텐츠 성격을 반영한 키워드 추천
- 키워드 추천 원칙:
  - 검색 노출을 고려한 실제 많이 찾는 단어
  - 채널명이나 설명에 사용된 핵심 단어와 연계
  - 너무 일반적이거나 길고 복잡한 키워드 피함
  - 최소 5개 이상, 최대 10개
  - 해외 대상자일 경우 해당 국가 언어로 추천
- 키워드 목록 제시
- 선택/수정 가능

### 데이터

- 추천된 키워드 목록 (5-10개)
- 선택된 키워드 목록

---

## Step 6: 프로필 이미지 생성 (Profile Image)

### 기능

- 채널 주제와 분위기를 반영한 프로필 이미지 생성
- 프로필 이미지 생성 원칙:
  - 유튜브 권장 규격: 800x800px (원형 표시 고려)
  - 중심에 메인 심볼 배치
  - 작은 크기에서도 식별 가능한 굵은 도형과 선명한 색상
  - 과도한 디테일, 복잡한 배경, 작은 텍스트 피함
  - 글자는 넣지 않고 색감을 뚜렷하고 심플하게
- 이미지 생성 및 미리보기
- 마음에 들지 않을 경우 재생성 가능
- 글자 추가가 필요한 경우:
  - 미리캔버스 사이트 안내
  - 또는 1:1 비율 배경 이미지 제공
  - ImageFX 활용 안내 및 프롬프트 변환 제공

### 데이터

- 생성된 프로필 이미지 (Base64 또는 URL)
- 이미지 생성 프롬프트

---

## 공통 기능

### 로딩 상태

- 각 단계별 로딩 스피너
- 버튼 비활성화 (로딩 중)
- 로딩 텍스트 표시

### 에러 처리

- Toast 메시지로 성공/실패 알림
- 콘솔 에러 로깅

### 세션 관리

- 세션 ID 기반 데이터 저장
- 단계별 진행 상태 저장
- 데이터 복원 (새로고침 시)

### 사용자 안내

- 각 단계별 친절한 설명 제공
- 격려하는 톤 유지
- "유훈 마스터 클래스 수강생 님"으로 지칭
- 창의적인 아이디어 브레인스토밍 지원

---

## 데이터 흐름

1. **세션 생성** (Step 1)
   - 채널 정보 입력 및 세션 생성

2. **데이터 저장**
   - 각 단계에서 세션 데이터 저장
   - 개별 필드로 저장

3. **AI API 호출**
   - OpenAI GPT: 채널명, 핸들명, 설명, 키워드 추천
   - Image Generation API: 프로필 이미지 생성

4. **외부 링크 제공**
   - 유튜브 검색 링크
   - 상표권 검색 사이트 (키프리스)
   - 미리캔버스 사이트
   - ImageFX 안내

---

## 주요 기능 요약

- 6단계 워크플로우: Channel Info → Channel Name → Handle → Description → Keywords → Profile Image
- 각 단계별 AI 기반 추천 기능
- 채널명 검증 단계 (유튜브 검색, 상표권 확인)
- 단계별 명시적 진행 (Next Step 버튼)
- 세션 기반 데이터 저장 및 복원
- 친절하고 격려하는 사용자 경험
