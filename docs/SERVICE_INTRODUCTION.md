# TubeInsight 서비스 소개

## 📌 서비스 개요

**TubeInsight**는 YouTube 영상을 활용한 개인화된 학습 플랫폼입니다. 인기 YouTube 영상을 학습 자료로 변환하고, AI 기반 개인화 기능을 통해 각 학습자에게 맞춤형 학습 경험을 제공합니다.

### 핵심 가치

- **YouTube 기반 학습**: 인기 YouTube 영상을 학습 자료로 활용
- **개인화된 미션**: 학습자별 맞춤형 미션 제공
- **자동 자막 추출**: YouTube 영상의 자막을 자동으로 추출하여 학습 자료 생성
- **커뮤니티 학습**: 학습자 간 공지사항 확인 및 학습 경험 공유

---

## 🎯 주요 기능

### 1. YouTube 영상 분석 및 수집

- 인기 YouTube 영상 자동 수집
- 채널별 영상 통계 분석 (조회수, 좋아요, 댓글 등)
- Outlier 영상 자동 필터링 (VPH, Outlier VPH 기준)
- 지역별 인기 영상 수집 (KR, US 등)

### 2. 자동 자막 추출

- YouTube 영상 자막 자동 추출
- 자동 생성 자막/수동 자막 구분
- 롱폼 및 숏폼 영상 지원

### 3. AI 기반 개인화 학습 자료

- Gemini API를 활용한 개인화된 학습 자료 생성
- 학습자 프로필 기반 맞춤형 콘텐츠 제공
- 채널 분석 및 영상 추천

### 4. 미션 관리 시스템

- 기수별 미션 생성 및 관리
- 학습자별 미션 제출 및 피드백
- 미션 진행 상황 추적

## 👨‍💼 관리자(Admin) 기능

### 1. 대시보드

- 핵심 지표 모니터링
- 최근 활동 추적
- 시스템 통계 시각화

### 2. 공지사항 관리

- 공지사항 작성, 수정, 삭제
- 카테고리별 공지사항 관리
- 중요 공지사항 고정 기능

### 3. 미션 관리

- 미션 공지 작성 및 관리
- 기수별 미션 할당
- 미션 제출 현황 확인
- 미션 피드백 제공

### 4. 수강생 관리

- 수강생 목록 조회 및 검색
- 수강생 상세 정보 확인
- 수강생 계정 관리
- 학습 진도 모니터링

### 5. 기수 관리

- 기수(Cohort) 생성 및 관리
- 기수별 수강생 할당
- 기수별 미션 관리
- 기수별 통계 확인

### 6. YouTube 관리

#### 6.1 채널 관리

- 수집된 YouTube 채널 목록 조회
- 채널별 통계 정보 확인
  - 평균 조회수 (overallAvgView)
  - 영상 수 (videoCount)
  - 마지막 수집 시간
- 지역별 필터링 (KR, US 등)
- 채널 검색 기능

#### 6.2 영상 관리

- 수집된 YouTube 영상 목록 조회
- 영상별 상세 정보 확인
  - 조회수 (viewCount)
  - 시간당 조회수 (VPH)
  - Outlier VPH (최근 평균 대비)
  - Outlier View (전체 평균 대비)
- 지역별 필터링
- 영상 검색 기능
- YouTube 링크 제공

### 7. 크론잡 관리

- 크론잡 실행 결과 모니터링(유튜브 채널, 영상들)

## 👨‍🎓 수강생(Student) 기능

### 1. 대시보드

- 학습 진행률 확인
- 최근 미션 현황
- 인기 YouTube 영상 추천
- 학습 통계 시각화

### 2. 학습

#### 2.1 내 강의

- 기수별 강의 목록
- 강의별 학습 진도 확인
- 강의 자료 다운로드

#### 2.2 강의 영상

- YouTube 기반 강의 영상 라이브러리
- 영상별 자막 및 학습 자료 제공
- 영상 시청 및 학습 진행 상황 추적

#### 2.3 학습 자료

- 제출한 학습 자료 목록
- 학습 자료 다운로드
- 학습 이력 확인

### 3. 과제

#### 3.1 나의 미션

- 기수별 미션 목록
- 미션 상세 정보 확인
- YouTube 영상 분석 및 자막 추출
- 개인화된 학습 자료 생성
- 미션 제출

#### 3.2 제출 내역

- 제출한 미션 목록
- 제출 상태 확인
- 제출 내용 수정

#### 3.3 성적 확인

- 미션별 점수 확인
- 피드백 확인
- 성적 통계

### 4. 커뮤니티

#### 4.1 공지사항

- 최신 공지사항 확인
- 중요 공지사항 확인
- 공지사항 검색

#### 4.2 Q&A

- 질문 작성 및 답변 확인
- 커뮤니티 질문 검색
- 답변 알림

### 5. 포인트 관리

- 포인트 충전
- 포인트 사용 내역
- 포인트 잔액 확인

### 6. 프로필 관리

- 개인 정보 수정
- 학습 목표 설정
- 학습 수준 설정
