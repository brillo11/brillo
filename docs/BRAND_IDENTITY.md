# 브랜드 아이덴티티 가이드

## 서비스 이름

### 추천: **LearnFlow** (러닝플로우)

**의미**:

- Learn (학습) + Flow (흐름)
- YouTube 영상의 흐름을 따라 학습하는 서비스
- 개인화된 학습 흐름을 제공한다는 의미

**대안**:

- **VideoLearn** (비디오런) - 직관적이지만 일반적
- **StudyFlow** (스터디플로우) - 학습 흐름 강조
- **EduTube** (에듀튜브) - 교육 + YouTube 조합

## 서비스 슬로건

**"YouTube로 시작하는 개인화된 학습"**

또는

**"나만의 학습 자료, YouTube에서"**

## 타이포그래피

- **Primary**: Pretendard (한글 최적화)

## 아이콘 시스템

- **라이브러리**: Lucide React
- **스타일**: Outline (기본), Fill (강조)
- **크기**: 16px, 20px, 24px, 32px

## 로고 컨셉

### 로고 아이디어

1. **플레이 버튼 + 학습 아이콘** 조합
2. **흐름을 나타내는 곡선 + L** 조합
3. **YouTube 플레이 버튼 스타일 + LearnFlow 텍스트**

### 로고 색상

- **Primary**: LearnFlow Blue (#3B82F6)
- **Accent**: YouTube Red (#FF0000) - 선택적 사용

## 디자인 원칙

1. **명확성**: 정보가 명확하게 전달되어야 함
2. **일관성**: 모든 페이지에서 일관된 디자인 언어 사용
3. **접근성**: WCAG 2.1 AA 기준 준수
4. **반응성**: 모든 디바이스에서 최적화된 경험 제공
5. **학습 친화적**: 학습에 집중할 수 있는 깔끔한 인터페이스
