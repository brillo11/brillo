# AI Assistant 페이지 기능명세서

## 개요

유튜브 영상 제작을 위한 AI 어시스턴트 페이지. 8단계의 워크플로우를 통해 제목, 썸네일, 대본, 메타데이터, 쇼츠 제목, 음성을 생성합니다.

## 전체 구조

- 상단에 8단계 진행 상황 표시 (Progress Bar)
- 단계: Persona → Topic → Title → Style → Design → Script → Metadata → Final (Shorts + Voice)

---

## Step 1: AI 파트너 선택 (Persona)

### 기능

- 4개의 페르소나 중 1개 선택
- 선택 후 "Next" 버튼 클릭 시 Step 2로 이동

### 데이터

- 4개의 페르소나 옵션 제공
- 기본 페르소나: "유튜브 훈련소 AI 크리에이터"

---

## Step 2: 주제 입력 (Topic)

### 기능

- 주제/토픽 텍스트 입력
- Enter 키로 제출 가능
- 로딩 중 입력 필드 비활성화
- 제출 시:
  - 세션 생성
  - 제목 자동 생성 (Step 3)
  - Step 3으로 자동 이동

### 상태

- 로딩 스피너 표시
- 성공/실패 토스트 메시지

---

## Step 3: 제목 선택 (Title)

### 기능

- 5개의 제목 세트 중 1개 선택
- 선택된 제목 하이라이트
- "Next Step" 버튼 클릭 시 Step 4로 이동

### 데이터

- 각 제목 세트: 썸네일 제목 + 영상 제목

---

## Step 4: 썸네일 가이드 선택 (Thumbnail Guide)

### 기능

- **YouTube 채널 연동** (선택 사항):
  - 채널 URL 입력
  - YouTube Data API v3를 통해 최근 영상의 썸네일 n개 가져오기 (기본 10개)
  - 가져온 썸네일 중 1개 선택 가능
  - 선택된 썸네일은 Step 5 썸네일 생성 시 참조 이미지로 사용
  - Gemini API의 컨텍스트 유지 기능 활용 (나노바나나)
- **썸네일 가이드 선택**:
  - 3개의 썸네일 가이드 중 1개 선택
  - "상세 확인하기" 토글 버튼으로 상세 정보 표시/숨김
- **진행**:
  - 가이드 선택 후 "Next Step" 버튼 클릭 시 Step 5로 이동
  - 진입 시 자동으로 가이드 생성 (없는 경우)
  - 선택된 YouTube 썸네일이 있으면 Step 5에서 자동으로 참조 이미지로 전달

### 데이터

- 썸네일 가이드: 제목, 요약, 상세 설명 (Markdown 형식)
- YouTube 썸네일: 채널 URL, 채널 ID, 썸네일 목록, 선택된 인덱스

---

## Step 5: 썸네일 생성 (Thumbnail Generation)

### 기능

- 진입 시 자동으로 썸네일 생성
  - Step 4에서 선택된 YouTube 썸네일이 있으면 참조 이미지로 자동 전달
  - Gemini API의 컨텍스트 유지 기능으로 채널 스타일 반영
- 썸네일 미리보기 (Base64 이미지)
- 텍스트 기반 수정 요청
- 참조 이미지 업로드 (선택)
  - 로컬 파일 업로드
  - Step 4에서 선택한 YouTube 썸네일 (자동 적용)
- 썸네일 수정 실행
- "Confirm" 버튼 클릭 시 Step 6으로 이동

### 상태

- 초기 생성 로딩 스피너
- 수정 중 로딩 상태

---

## Step 6: 대본 생성 (Script)

### 기능

- 진입 시 자동으로 대본 생성
- Markdown 형식 대본 렌더링
- 대본 복사 기능
- "Next Step" 버튼 클릭 시 Step 7로 이동

### 데이터

- 대본 텍스트 (Markdown 형식)
- 썸네일 이미지 URL
- 선택된 제목
- 토픽

---

## Step 7: 메타데이터 생성 (Metadata)

### 기능

- 진입 시 자동으로 메타데이터 생성
- 각 항목별 복사 기능:
  - 설명 (300자 내외 요약)
  - 타임스탬프 (시간별 제목 목록)
  - 해시태그 (4개)
  - 태그 (최대 7개)
- "Next Step" 버튼 클릭 시 Step 8로 이동

### 데이터

- 설명, 타임스탬프 배열, 해시태그 배열, 태그 배열

---

## Step 8: 마무리 (Final - Shorts Titles + Voice Generation)

### 기능

- **쇼츠 제목 생성**:
  - 진입 시 자동으로 쇼츠 제목 생성
  - 각 챕터별 2~3개의 쇼츠 제목 제공
  - 개별 제목 클릭 시 복사
  - 챕터별 전체 복사
- **음성 생성**:
  - **목소리 등록**:
    - 마이크 녹음 기능 (실시간 녹음)
    - 녹음된 음성 파일 업로드
    - 일레븐랩스 API를 통한 목소리 클론 생성
    - 목소리 클론 완료 후 확인
  - **대본 음성 합성**:
    - Step 6에서 생성된 대본을 선택한 목소리로 변환
    - 일레븐랩스 API를 통한 음성 합성
    - 생성된 음성 미리듣기
    - 음성 다운로드 기능

### 데이터

- 챕터별 쇼츠 제목 배열 (각 챕터당 2~3개)
- 목소리 클론 ID (일레븐랩스)
- 녹음된 음성 파일 (Base64 또는 URL)
- 생성된 음성 파일 (Base64 또는 URL)
- 대본 텍스트

---

## 공통 기능

### 로딩 상태

- 각 단계별 로딩 스피너
- 버튼 비활성화 (로딩 중)
- 로딩 텍스트 표시

### 에러 처리

- Toast 메시지로 성공/실패 알림
- 콘솔 에러 로깅

### 세션 관리

- 세션 ID 기반 데이터 저장
- 단계별 진행 상태 저장
- 데이터 복원 (새로고침 시)

### 자동 생성

- Step 4: 썸네일 가이드 자동 생성
- Step 5: 썸네일 자동 생성
- Step 6: 대본 자동 생성
- Step 7: 메타데이터 자동 생성
- Step 8: 쇼츠 제목 자동 생성

---

## 데이터 흐름

1. **세션 생성** (Step 2)
   - `createAIAssistantSession()` 호출
   - `conversationId` 생성 (OpenAI)

2. **데이터 저장**
   - 각 단계에서 `updateAIAssistantSession()` 호출
   - 개별 필드로 저장 (JSON 필드 아님)

3. **AI API 호출**
   - OpenAI GPT-5.1: 제목, 가이드, 대본, 메타데이터, 쇼츠 제목
   - Google Gemini: 썸네일 이미지 생성
   - YouTube Data API v3: 채널 정보 및 썸네일 가져오기
   - ElevenLabs API: 목소리 클론 및 음성 합성

4. **컨텍스트 유지**
   - Step 4에서 선택한 YouTube 썸네일을 Step 5로 전달
   - Gemini API의 컨텍스트 유지 기능(나노바나나) 활용
   - 참조 이미지와 가이드를 결합하여 일관된 스타일 생성

5. **음성 생성 프로세스** (Step 8)
   - 마이크 녹음 또는 음성 파일 업로드
   - 일레븐랩스 API로 목소리 클론 생성
   - 생성된 대본을 클론된 목소리로 음성 합성
   - 음성 파일 다운로드 및 재생

---

## 주요 기능 요약

- 8단계 워크플로우: Persona → Topic → Title → Thumbnail Guide → Thumbnail → Script → Metadata → Final (Shorts + Voice)
- 각 단계별 자동 생성 기능
- YouTube 채널 썸네일 참조 기능 (Step 4)
- Gemini 컨텍스트 유지 기능으로 채널 스타일 반영
- 일레븐랩스 음성 생성 기능 (Step 8 - 마무리 단계)
- 단계별 명시적 진행 (Next Step 버튼)
- 세션 기반 데이터 저장 및 복원
